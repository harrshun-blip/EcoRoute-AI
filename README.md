# EcoRoute AI

EcoRoute AI is an AI-powered Sustainable Transport Advisor designed to support SDG 11 (Sustainable Cities and Communities) and SDG 13 (Climate Action).

## Features

- Compare transportation modes
- Estimate carbon emissions
- Recommend sustainable travel options
- Promote eco-friendly commuting

## Technology Stack

- Python
- Streamlit
- Pandas

## SDG Alignment

- SDG 11: Sustainable Cities and Communities
- SDG 13: Climate Action
